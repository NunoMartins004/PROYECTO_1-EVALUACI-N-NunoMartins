# Juego caza del Tesoro

## Descripción
El jugador controla a un héroe que debe moverse por una cuadrícula de 10x10 hasta llegar a un cofre. 
El movimiento del personaje se realiza mediante tiradas de un dado.

## Funcionalidades
- Validación del nombre del jugador.
- Generación dinámica del tablero 10x10.
- Movimiento del héroe controlado por un dado.
- Resaltado de celdas válidas para moverse.
- Contador de tiradas realizadas.
- Guardado y comparación del récord mediante localStorage.

# Tecnologías usadas
- HTML
- CSS
- JavaScript

## Cómo ejecutar el proyecto
1. Descargar el proyecto o clonar el repositorio.
2. Abrir el archivo "index.html" en un navegador web.
